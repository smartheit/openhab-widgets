# openHAB Main UI Widgets by smartheit

This repository provides custom-built widgets for Main UI,
being used for projects at [smartheit GmbH](https://smartheit.com/).

## Compatibility

- Widgets are designed to be working with the openHAB 4.3.x series.

## Widgets

### Configuration

Configuration widgets are being used where the user needs to configure the system through Items.

Currently, the following configuration widgets exist:

- [`parameter-list`](configuration/parameter-list.yaml): A widget implementing a list configuration parameter, where the list entries are serialised to a String Item by joining them with a comma.
